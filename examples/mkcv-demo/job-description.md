# Senior Backend Engineer -- Platform Team

**Company:** NovaCraft Technologies
**Location:** San Francisco, CA (Hybrid -- 3 days in office)
**Employment Type:** Full-time

---

## About NovaCraft

NovaCraft Technologies is a Series C fintech startup building the next
generation of payment infrastructure for global e-commerce. Our platform
processes $4B+ in annual transaction volume across 30 countries, and we're
growing 3x year-over-year. We're a team of 180 (engineering team of 60)
backed by top-tier investors.

## About the Role

We're looking for a Senior Backend Engineer to join our Platform team. This
team owns the core services that every product team at NovaCraft depends on:
the API gateway, event bus, authentication layer, and shared service libraries.

You'll design and build high-throughput, low-latency systems that process
millions of transactions per day. You'll work closely with product engineering
teams to define platform contracts and ensure our infrastructure scales with
the business. This is a high-impact IC role with significant architectural
ownership -- you'll shape the technical direction of our platform for the next
3-5 years.

## What You'll Do

- Design, build, and operate backend services handling 30,000+ requests per
  second with sub-100ms p99 latency requirements
- Own the architecture and roadmap for our internal API gateway and service
  mesh infrastructure
- Build event-driven data pipelines for real-time transaction processing
  and fraud detection
- Define and enforce API standards, service contracts, and observability
  practices across 15+ product engineering teams
- Collaborate with Security and Compliance teams to maintain PCI-DSS and
  SOC 2 compliance across all platform services
- Mentor mid-level engineers and contribute to hiring (resume review,
  technical interviews)
- Write technical design documents and ADRs for major architectural decisions
- Participate in on-call rotation for platform services (1 week per quarter)

## Requirements

### Must Have

- 6+ years of professional software engineering experience
- Strong proficiency in Python or Go (we use both)
- Deep experience with distributed systems (event-driven architectures,
  message queues, service-to-service communication)
- Production experience with Kubernetes and container orchestration
- Solid understanding of relational databases (PostgreSQL preferred) at
  scale -- query optimization, replication, partitioning
- Experience with event streaming platforms (Kafka, Pulsar, or similar)
- Track record of designing and operating services at 10,000+ RPS
- Strong written and verbal communication skills

### Strongly Preferred

- Experience with API gateway design and service mesh (Envoy, Istio, Kong)
- Infrastructure-as-code experience (Terraform, Pulumi, or CloudFormation)
- Familiarity with fintech or payment systems
- Experience mentoring engineers and leading technical design reviews
- Contributions to open-source projects

### Nice to Have

- Experience with gRPC and Protocol Buffers
- Familiarity with Rust or other systems programming languages
- Experience building internal developer platforms or SDKs
- Background in machine learning model serving infrastructure

## Tech Stack

- **Languages:** Python, Go
- **Infrastructure:** AWS (EKS, RDS, SQS, Lambda), Kubernetes, Terraform
- **Data:** PostgreSQL, Redis, Apache Kafka, DynamoDB
- **Observability:** Datadog, OpenTelemetry, PagerDuty
- **CI/CD:** GitHub Actions, ArgoCD
- **Communication:** gRPC (internal), REST (external APIs)
